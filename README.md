# React + TypeScript + Vite

This app is made using React TypesScript and Vite

## Getting started

## Step 1: Install dependencies

```bash
# using npm
npm install --force

# OR using Yarn
yarn install --force
```

## Step 2: Run code

### Dev Mode

```bash
# using npm
npm run dev

# OR using Yarn
yarn run dev
```

### Build Mode

```bash
# using npm
npm run build

# OR using Yarn
yarn run build
```

### Vite Preview Mode

You need to build your project first

```bash
# using npm
npm run preview

# OR using Yarn
yarn run preview
```
